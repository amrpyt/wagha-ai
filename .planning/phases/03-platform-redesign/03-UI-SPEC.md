# Phase 03 — UI Design Contract

> Visual and interaction contract for Phase 03: platform-redesign. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none |
| Preset | not applicable |
| Component library | none (native HTML + Tailwind CSS v4) |
| Icon library | inline SVG (Lucide-style, embedded directly in components) |
| Font | Noto Sans Arabic (Google Fonts, via `next/font/google`) |
| RTL approach | HTML `dir="rtl"` + Tailwind `rtl:` modifier + logical properties |
| State management | React `useState` + `useCallback` (no Redux/Zustand) |

**Design system established by:** existing codebase survey (Grep scan of all components)

## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| 0.5 | 2px | Icon internal spacing, checkbox/checkbox accent |
| 1 | 4px | Inline element gaps, tight stacks |
| 2 | 8px | Compact element spacing (ModifierPicker mood buttons gap-2) |
| 3 | 12px | Between related elements within a section |
| 4 | 16px | Default element padding, section gaps |
| 5 | 20px | Panel header padding |
| 6 | 24px | Section-level padding |
| 8 | 32px | Layout gaps between sidebar and main |
| 12 | 48px | Major section breaks |
| 16 | 64px | Page-level top/bottom spacing |

**Exceptions:** none — all phase components use 4-point scale.

## Typography

| Role | Size | Weight | Line Height | Notes |
|------|------|--------|-------------|-------|
| Body | 14px | 400 (normal) | 1.5 | Default text, labels in `OptionSelect` |
| Label | 12px | 500 (medium) | 1.4 | `ModifierPicker` section labels (`text-xs text-gray-500`) |
| Small | 12px | 400 | 1.4 | Version strip labels, muted hints |
| Heading | 16px | 600 (semibold) | 1.3 | CollapsibleSection titles, breadcrumb active |
| Display | 20px | 600 | 1.3 | Modal headings |

All text is Arabic. `text-right` for RTL alignment. Font: Noto Sans Arabic.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (background) | `#FAFAFA` | Dashboard layout bg, page background |
| Secondary (surfaces) | `#FFFFFF` (white) | Cards, panels, sidebar, inputs, buttons |
| Accent (10%) | `#1E3A5F` (navy) | Primary buttons, active tabs, focus rings, selected states, links |
| Destructive | `#EF4444` (red-500) | Error state backgrounds (`bg-red-50`), validation messages |

**Accent reserved for:**
- Primary CTA button background (`bg-[#1E3A5F]`)
- Active/inactive tab differentiation in `RenderTypeTabs`
- Selected pill state in `TemplatePills`
- Focus rings on all interactive inputs (`focus:ring-[#1E3A5F]`)
- Hover border color on interactive elements (`hover:border-[#1E3A5F]`)
- Links and link-style text (`hover:text-[#1E3A5F]`)
- Accent checkbox (`accent-[#1E3A5F]`)
- Breadcrumb active state

**NOT accent:** destructive actions (use red-500), success states (use contextual greens in ModifierPicker mood options)

**Semantic color usage in existing components:**
- Mood picker option backgrounds: `bg-gray-100` (neutral), `bg-green-100` (vibrant), `bg-blue-100` (calm), `bg-purple-100` (dramatic)
- Error: `bg-red-50 border-red-200` with `text-red-700`
- Disabled: `bg-gray-300 text-gray-500`

## Layout

**Structure:** Fixed left panel + scrollable main content area

```
┌─────────────────────────────────────────────────────────────────┐
│ TopBar (brand_color adaptive)                                    │
├────────────┬────────────────────────────────────────────────────┤
│            │ Breadcrumb: لوحة التحكم / مشروع جديد               │
│  LeftPanel │ RenderTypeTabs: [خارجي AI] [داخلي AI]            │
│  (288px)   │ TemplatePills: [حديث] [كلاسيكي] ...              │
│            │ ┌──────────────────────────────────────────────┐   │
│  Collaps-  │ │ Canvas Area (520px height)                  │   │
│  ible      │ │ States: empty → image-loaded → generating   │   │
│  Sections  │ │           → complete (BeforeAfterSlider)     │   │
│            │ └──────────────────────────────────────────────┘   │
│  Camera    │ PromptInput (textarea)                            │
│  Surrounds │ ReferenceImages (up to 4 slots)                   │
│  Lighting  │ [Error banner if present]                         │
│  Ground    │ [Save Preset] [توليد — GenerateButton]            │
│  Annotations│ Version Strip (if hasVersions)                   │
└────────────┴────────────────────────────────────────────────────┘
```

**LeftPanel specs:**
- Width: `w-72` (288px), `flex-shrink-0`
- Background: white, `border border-gray-100 rounded-xl`
- Max height: `calc(100vh - 140px)`, `overflow-y-auto`
- Sections: `CollapsibleSection` with emoji title, `border-b border-gray-100`
- Section padding: `px-4`, inner content `pb-3 px-1 space-y-3`

**Canvas Area specs:**
- Height: fixed `520px`
- Background: white, `border border-gray-200 rounded-xl overflow-hidden`
- Top action bar: `flex items-center justify-between px-4 py-3 border-b border-gray-100`

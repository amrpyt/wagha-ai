**1. Add `selectiveEdit` to `RenderOptions` interface:**
```typescript
export interface RenderOptions {
  imageBuffers: Buffer[]
  referenceBuffers?: Buffer[]
  renderType: RenderType
  template: ExteriorTemplate | InteriorTemplate
  modifiers: RenderModifiers
  customPrompt?: string
  selectiveEdit?: boolean  // NEW: D-12/D-13/D-14
  signal?: AbortSignal
  onProgress?: (progress: number, status: string) => void
}
```

**2. Update `buildPrompt` to append selective editing instruction when true:**

In the `buildPrompt` function, before the closing:
```typescript
// Selective editing mode — preserve original, only change specified element
if (options.selectiveEdit && customPrompt?.trim()) {
  parts.push(
    'Keep the original image exactly as-is. Only change the following element as specified. Do not alter anything else: ' +
    customPrompt.trim() +
    '. Preserve all other architectural elements, materials, colors, and composition unchanged.'
  )
}
```

The key logic: when `selectiveEdit=true` AND there is a custom prompt, inject a preservation instruction that tells the AI to keep everything identical EXCEPT the specified change.

**3. Update `/api/generate` route to pass selectiveEdit from project to buildPrompt:**

Read `/api/generate/route.ts` first. The route reads the project from DB. Add:
```typescript
// Read selectiveEdit from project
const selectiveEdit = project.selective_edit ?? false

// Pass to buildPrompt

  imageBuffers: [processedBuffer],
  renderType: project.render_type as RenderType,
  template: project.template as ExteriorTemplate | InteriorTemplate,
  modifiers: JSON.parse(project.modifiers || '{}'),
  customPrompt: project.custom_prompt || undefined,
  selectiveEdit,  // NEW
  signal: abortController.signal,
})
```

This wires the entire chain: UI checkbox -> FormData -> project record -> generate endpoint -> buildPrompt.
